What is the "NOT" Operator?

The "NOT" operator reverses the truth value of a condition. True becomes False, False becomes True.

It is widely used in programming and Boolean logic to exclude specific results or conditions.

Purpose of the "NOT" Operator

Negates or inverts a condition’s truth value. Helps filter unwanted results or ensure certain conditions are not true.

Examples

  • Boolean logic: True → NOT True = False
  • False → NOT False = True

Programming (Python):

x = 5
if not (x > 10):
    print("x is NOT greater than 10")

Search example: cats NOT dogs shows cats results excluding dogs.
